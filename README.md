## config

This directory contains configuration files for coding agents and skills. Each file includes predefined rules, descriptions, and settings to help streamline the setup and usage of various coding agents and skills.

### Files

- `.claude/CLAUDE.md`: Configuration for the Claude coding agent, including foundational rules, relationship guidelines, and coding standards.
- `.local.md`: Local configuration settings specific to the user's environment and preferences.

### Usage

To utilize these configurations, copy the desired file into your coding agent or skill's configuration directory.

### Customization

Feel free to modify the configurations to better suit your workflow and project requirements. Ensure that any changes align with your overall coding standards and practices.
